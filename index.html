PROJECTS

  • Full Stack Instagram Clone

    This full stack Instagram clone is a social media application that replicates the core features of Instagram. The application allows users to create and manage their profiles, follow and unfollow other users, upload and edit pictures and videos, view and like posts, and comment on them. It also includes features such as user authentication, authorization, and data validation, providing a secure and reliable user experience. - Still in development! -


    The application is developed using a range of technologies, including HTML, CSS, Javascript, ReactJS, Node.js, Express.js, and MongoDB.
  • Google Trends website

    The Google Trends website offers an application that provides insights into the latest trends, searches, and stories from around the world. With this website, you can search for specific data on any term and view its statistics in every (supported) country, see different time ranges, and compare it with other searches.


    This website is built using a variety of tools, including HTML, SCSS/SASS, Javascript, Axios, AJAX, ReactJS, ContextAPI, and Serverless functions, all hosted on Netlify-lambda.
  • To-Do list

    The To-Do list application enables you to create and manage tasks across different boards. You can set and change deadlines, color code your tasks, add additional information, rename, delete and re-order them. This application utilizes Redux to manage state data across a large component tree, resulting in cleaner and more readable code, and faster performance.


    This application is developed using HTML, CSS, SCSS/SASS, Javascript, ReactJS, and Redux.
  • Recipe website

    Discover millions of recipes online with the option to specify food categories such as vegan, vegetarian, kosher, and more. This website is powered by a free recipe REST API, allowing you to search for recipes based on your preferences. Note that due to API limitations, please limit your entries to 10 per minute.


    The recipe website is built using HTML, CSS, SCSS/SASS, Javascript, ReactJS, AJAX, and REST API.
