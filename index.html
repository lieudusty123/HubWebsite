PROJECTS

  • Google Trends website

    Google trends application, which will show you the current stories, trends, and searches from around the globe. The website, let you search for specific data on every term you would like. You can see the term's statistics in every single (supported) country, see different time ranges, and also compare it with other searches.

    Made with HTML, SCSS/SASS, Javascript, Axios, AJAX, ReactJS, ContextAPI, Serverless functions, Netlify-lambda.
  • To-Do list

    An application which lets you create tasks inside different boards. You can set / change deadlines for a specific task item, color it, set more info, rename, delete and re-order it. Used redux to manage state data over a large component tree to achieve a faster and more readable, cleaner code.

    Made with HTML, CSS, SCSS/SASS, Javascript, ReactJS, Redux.
  • Recipe website

    Search millions of recipes online with the option to specify food cateogries (vegan, vegeterian, kosher etc.). Built with the help of an online free recipe REST API. (since this website uses a free API, please limit your entries to 10 per minute).

    Made with HTML, CSS, SCSS/SASS, Javascript, ReactJS, AJAX, REST API.
  • Wordle game clone

    You have five chances to guess a random five-letter word. With each guess, the game tells you which letters are correct, which are in the wrong spots, and which are not in the word at all.

    Made with HTML, CSS, Javascript.
